React component, which can be used to drag and resize, with location limits.

It's easy!! Try it！！

## Install

```bash
npm install react-drag-resizable --save
```

or

```bash
yarn add react-drag-resizable
```

## Usage

```js
import { DragResizableBox } from 'react-drag-resizable';
```
